# Usage
<!-- usage -->
```sh-session
$ npm install -g DownloaderCLI
$ DownloaderCLI COMMAND
running command...
$ DownloaderCLI (-v|--version|version)
DownloaderCLI/0.0.1 win32-x64 node-v11.6.0
$ DownloaderCLI --help [COMMAND]
USAGE
  $ DownloaderCLI COMMAND
...
```
